## Decisions Made

- **Imports outside include guard:** Julia's macro resolution requires `using Printf` etc. to be visible at parse time. Moving `using` statements before `if !(@isdefined _PHYSICS_INSIGHT_JL_LOADED)` fixes `@sprintf not defined` error when included from another script.
- **Zero-fill noise floor bins:** Setting `phi_norm[!signal_mask] = 0` (rather than NaN or leaving random) prevents random noise-floor phase from distorting y-axis scale in the overlay figures.
- **Polygon fit in rad/s domain:** `omega_rad_s = 2pi * df_THz * 1e12` converts THz offset to rad/s, matching `decompose_phase_polynomial`'s internal convention so the polynomial overlay is consistent with the decomposition.
