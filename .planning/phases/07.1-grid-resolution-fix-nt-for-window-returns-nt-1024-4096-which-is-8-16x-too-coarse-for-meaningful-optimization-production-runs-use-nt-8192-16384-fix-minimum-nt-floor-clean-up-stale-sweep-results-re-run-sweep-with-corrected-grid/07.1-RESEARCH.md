### scripts/common.jl

**nt_for_window** (lines 226-234): Returns smallest power-of-2 >= ceil(time_window_ps / dt_min_ps). Default dt_min=0.0105 ps. For time_window=5 ps: ceil(5/0.0105)=477, next pow2=512. For time_window=10 ps: 1024. For time_window=86 ps (L=2m, P=0.3W, safety=3): 8192. The function is correct; it's the caller that needs the floor.
