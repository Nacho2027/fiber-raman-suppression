**New function: `pr_optimize_and_propagate(preset, P_cont, L_target; max_iter=50, n_zsave=PR_N_ZSAVE)`** (per D-07):
1. Call `setup_raman_problem(L_fiber=L_target, P_cont=P_cont, beta_order=3, fiber_preset=preset)`. For L_target >= 5m: pass explicit `Nt=65536, time_window=500.0` (SMF-28) or `Nt=65536, time_window=463.0` (HNLF). For L_target <= 2m: let auto-sizing work.
2. Call `optimize_spectral_phase(uω0, fiber, sim, band_mask; max_iter=max_iter, log_cost=true)`.
3. Extract phi_opt: `reshape(Optim.minimizer(result), sim["Nt"], sim["M"])`.
4. Propagate shaped through 2 * L_target with z-saves (100 points from 0 to 2*L_target).
   - `deepcopy(fiber)`, set `fiber_2x["L"] = 2*L_target`, `fiber_2x["zsave"] = collect(LinRange(0, 2*L_target, n_zsave))`.
   - Update dispersion operator: `fiber_2x["Dω"]` does not depend on L, so no change needed.
   - `uω0_shaped = uω0 .* exp.(1im .* phi_opt)`
   - `sol = MultiModeNoise.solve_disp_mmf(uω0_shaped, fiber_2x, sim)`
5. Also propagate flat phase through same 2*L_target.
6. Compute J(z) for both. Extract J at z=L_target (the optimization point) and at z=2*L_target (the decay point).
7. Return named tuple: phi_opt, J_z_shaped, J_z_unshaped, zsave, J_at_target, J_at_2x, J_flat_at_target, J_flat_at_2x, Nt, tw, converged, n_iter, grad_norm.

**New function: `pr_find_horizon(J_z_dB, zsave, threshold_dB)`**:
Find z at which J_z_dB first exceeds threshold_dB. Use linear interpolation between z-points for sub-grid precision. Return NaN if threshold never crossed.

**New function: `pr_run_segmented_optimization(preset, P_cont; n_segments=4, L_segment=2.0, max_iter=50)`** (per D-09, D-10):

1. Setup a SINGLE grid sized for the segment length:
   ```julia
   uω0, fiber_base, sim, band_mask, Δf, _ = setup_raman_problem(
       L_fiber=L_segment, P_cont=P_cont, beta_order=3, fiber_preset=preset
   )
   ```
   For L_segment=2m at P=0.2W SMF-28: auto-sizing gives Nt=8192, tw=40ps — fine.

2. Initialize: `current_field = uω0` (fresh sech^2 pulse).

3. For each segment (1 to n_segments):
   a. **Optimize**: `result = optimize_spectral_phase(deepcopy(current_field), deepcopy(fiber_base), sim, band_mask; max_iter=max_iter, log_cost=true)`. Note: deepcopy the fiber because optimize_spectral_phase sets fiber["zsave"]=nothing.
   b. Extract phi_seg: `reshape(Optim.minimizer(result), sim["Nt"], sim["M"])`.
   c. **Propagate with z-saves** (25 per segment to keep total ~100 across all segments):
      - `fiber_seg = deepcopy(fiber_base); fiber_seg["zsave"] = collect(LinRange(0, L_segment, 25))`
      - `uω0_shaped = current_field .* exp.(1im .* phi_seg)`
      - `sol = MultiModeNoise.solve_disp_mmf(uω0_shaped, fiber_seg, sim)`
   d. **Compute J(z)** for this segment: `Float64[spectral_band_cost(sol["uω_z"][i,:,:], band_mask)[1] for i in 1:25]`
   e. **Extract output field** for next segment (lab frame):
      ```julia
      # ODE state is in interaction picture; transform back to lab frame
      Dω = fiber_base["Dω"]
      L = fiber_base["L"]
      current_field = cis.(Dω .* L) .* sol["ode_sol"](L)
      ```
      This must be a Matrix (Nt, M). Verify shape matches `size(uω0)`.
   f. **Boundary check**: `check_boundary_conditions(current_field, sim)`. If frac > 0.01, log warning — the field may have dispersed beyond the time window.
   g. Store: phi_seg, J_z_seg, J_seg_start, J_seg_end, bc_frac.

4. **Also run single-shot comparison** (per D-11):
   - Optimize for full L_total = n_segments * L_segment in a single optimization call.

   - Propagate with 100 z-saves.
   - Compute J(z).

5. **Also run flat phase (no optimization)** through full L_total with same grid as single-shot. 100 z-saves.

6. **Save results** to `results/raman/phase12/segmented_optimization.jld2`:

   - Metadata: n_segments, L_segment, L_total, P_cont, preset, Nt per condition.

Run for SMF-28 at P=0.2W (the canonical case). Expected wall time: ~5 minutes (4 segment optimizations at ~30s each + single-shot at ~200s).
