# Plan

## Objective

Define a clean mini-note series for the repo's major research paths, a shared
note template, and the missing-input work needed before note-writing agents can
execute efficiently.

## Recommended structure

1. Inventory the major research paths from active docs, research scripts, and
   current result artifacts.
2. Group paths into main notes versus sections inside a broader note.
3. Define one shared LaTeX template that keeps notes technically serious but
   short.
4. For each note target, identify:
   - core question
   - math delta
   - implementation surface
   - representative runs and figures
   - missing summaries or missing artifacts
5. Sequence later note-writing work so agents start with the strongest,
   best-documented lanes.

## Output destination

- Main recommendation lives in `SUMMARY.md`.
- This topic is internal planning only; no human-facing docs are written in this
  pass.
