# Cost Convention

The Raman objective is reported as Raman-band energy relative to total energy.
Lower is better.

## dB reporting

Use:

```text
J_dB = 10 log10(J)
```

A more negative value means less Raman-band leakage.

## Comparisons

When comparing two runs, say both the absolute values and the difference:

```text
phase-only:      -40.79 dB
amp-on-phase:    -46.91 dB
improvement:       6.12 dB
```

Avoid phrases like "higher score" unless the sign convention is restated.
