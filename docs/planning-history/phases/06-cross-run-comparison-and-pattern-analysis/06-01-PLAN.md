**Function 1: `compute_soliton_number(gamma_Wm, P0_W, fwhm_fs, beta2_s2m)` — per D-05**

Pure computation, no plotting. Returns Float64.
```
N = sqrt(gamma * P_peak * T0^2 / |beta2|)
```

**Function 2: `decompose_phase_polynomial(phi_opt, uomega0, sim_Dt, Nt)` — per D-04 / PATT-01**

Decomposes optimal phase onto GDD/TOD polynomial basis in the signal-bearing spectral region.

Algorithm (from Research Pattern 3):
1. Build signal-band mask from spectral power at -40 dB threshold (matching existing `_spectral_signal_xlim` convention)
   - `spec_power = abs2.(fftshift(uomega0[:, 1]))`
   - `dB = 10 .* log10.(spec_power ./ maximum(spec_power) .+ 1e-30)`
   - `signal_mask = dB .> -40.0`
2. Build angular frequency grid: `omega_shifted = 2pi .* fftshift(fftfreq(Nt, 1 / sim_Dt))` in rad/s
3. Extract signal-band phase: `phi_shifted = fftshift(phi_opt[:, 1])`, then `phi_signal = phi_shifted[signal_mask]`
4. Remove global offset and linear term (group delay normalization — CRITICAL per Pitfall 1 in Research):
   - `A_linear = hcat(ones(length(omega_signal)), omega_signal)`
   - `coeffs_linear = A_linear \ phi_signal`
   - `phi_detrended = phi_signal .- (coeffs_linear[1] .+ coeffs_linear[2] .* omega_signal)`
5. Fit 2nd+3rd order polynomial to detrended phase:
   - `A_poly = hcat(omega_signal.^2 ./ 2, omega_signal.^3 ./ 6)`
   - `coeffs_poly = A_poly \ phi_detrended`
6. Convert to physical units:
   - `gdd_fs2 = coeffs_poly[1] * 1e30` (rad*s^2 -> fs^2)
   - `tod_fs3 = coeffs_poly[2] * 1e45` (rad*s^3 -> fs^3)
7. Compute residual fraction: `norm(phi_detrended .- A_poly * coeffs_poly) / (norm(phi_detrended) + 1e-30)`

Returns a NamedTuple: `(gdd_fs2=..., tod_fs3=..., residual_fraction=...)`

Add docstring explaining what GDD/TOD mean physically (group delay dispersion, third-order dispersion), the unit conversions, and why offset+linear removal is necessary.

**Function 5: `plot_spectral_overlay(runs_fiber_group, fiber_type_label; save_path=nothing)` — per D-03 / XRUN-04**

Overlays optimized OUTPUT spectra for runs of the same fiber type on shared dB axes. This requires re-propagation.

- `runs_fiber_group`: Vector of Dicts for runs of ONE fiber type (e.g., all SMF-28 or all HNLF), each containing JLD2 fields

1. Reconstruct sim dict: Call `MultiModeNoise.get_disp_sim_params(run["lambda0_nm"] * 1e-9, 1, run["Nt"], run["time_window_ps"], 3)` — NOTE: `lambda0_nm` is in nm, need to convert to meters. M=1 for SMF. beta_order=3 per Phase 4 decision.

4. Apply optimal phase: `uomega0_shaped = @. run["uomega0"] * cis(run["phi_opt"])`
5. Propagate: `sol = MultiModeNoise.solve_disp_mmf(uomega0_shaped, fiber, sim)`

7. Compute power spectral density: `spec_out = abs2.(fftshift(uomega_out[:, 1]))`
