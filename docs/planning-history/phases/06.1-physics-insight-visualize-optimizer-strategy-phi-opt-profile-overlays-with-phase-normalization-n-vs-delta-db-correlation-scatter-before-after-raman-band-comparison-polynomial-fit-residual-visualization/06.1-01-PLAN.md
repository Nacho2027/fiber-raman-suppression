**Section 1 — Helper functions:**

`normalize_phase(phi_raw_fftorder, uomega0_fftorder, sim_Dt_ps, Nt)`:
- Takes raw (FFT-order) phi_opt and uomega0, both 1D vectors of length Nt.
- Builds signal mask at -40 dB on fftshifted `abs2.(fftshift(uomega0))`.
- Builds frequency offset grid: `df_THz = fftshift(fftfreq(Nt, 1.0 / sim_Dt_ps))` — this gives THz directly since sim_Dt is in ps.
- fftshifts phi_raw.
- Fits `a + b*omega` (in rad/THz: `omega = 2pi * df_THz`) over the signal-band and subtracts it from the full array.
- Zeros noise-floor bins (where `!signal_mask`) for clean display.
- Returns named tuple: `(phi_norm=..., df_THz=..., signal_mask=...)`.

`run_label(run)`:
- Returns `@sprintf("%s L=%.0fm P=%.2fW N=%.1f%s", run["fiber_name"], run["L_m"], run["P_cont_W"], run["soliton_number_N"], run["converged"] ? "" : " (nc)")`.

`build_lambda_axis_nm(df_THz, lambda0_nm)`:
- Computes absolute freq: `f_abs = C_NM_THZ / lambda0_nm .+ df_THz`.
- Selects positive frequencies: `pos = f_abs .> 0`.
- Returns `(lambda_nm = C_NM_THZ ./ f_abs[pos], pos_mask = pos)`.
