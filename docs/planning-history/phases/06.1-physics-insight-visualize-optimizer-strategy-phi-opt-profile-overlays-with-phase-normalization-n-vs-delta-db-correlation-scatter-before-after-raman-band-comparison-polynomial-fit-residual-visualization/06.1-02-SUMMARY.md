- **Global P_peak_global normalization:** Maximum input power across all runs used as dB reference in Fig 5 so all 5 subplots share a consistent dB scale.
- **NaN masking for group delay:** `gd_ps_masked[!signal_mask] = NaN` prevents noise-floor phase derivatives from corrupting the display, matching the zero-fill pattern used for phi_norm but using NaN (which matplotlib skips when plotting lines).
