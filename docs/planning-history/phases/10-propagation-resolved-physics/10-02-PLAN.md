From 10-RESEARCH.md Pattern 2 (band zeroing window):
```julia
function make_ablation_window(fs_fftshifted, band_lo, band_hi; sg_order=6, sg_width_frac=0.1)
    window = ones(length(fs_fftshifted))
    band_width = band_hi - band_lo
    sg_sigma = sg_width_frac * band_width
    for (i, f) in enumerate(fs_fftshifted)
        if band_lo < f < band_hi
            dist_lo = f - band_lo
            dist_hi = band_hi - f
            margin = min(dist_lo, dist_hi)
            window[i] = 1.0 - exp(-(margin / sg_sigma)^sg_order)
        end
    end
    return window
end
```

function shift_phase_spectrum(phi_opt, sim, delta_f_THz)

**Helper function: `pab_propagate_and_cost`**
Propagate with a given phi and return J at fiber end (no zsave for speed):
```julia
function pab_propagate_and_cost(uω0, phi, fiber, sim, band_mask)
    uω0_mod = uω0 .* exp.(1im .* phi)
    fiber_prop = deepcopy(fiber)
    # fiber["zsave"] is already nothing from setup_raman_problem
    sol = MultiModeNoise.solve_disp_mmf(uω0_mod, fiber_prop, sim)
    # Extract field at z=L from ODE solution
    uωf = cis.(fiber["Dω"] * fiber["L"]) .* sol["ode_sol"](fiber["L"])
    J, _ = spectral_band_cost(uωf, band_mask)
    return J
end
```

IMPORTANT: When `fiber["zsave"]` is `nothing`, `solve_disp_mmf` returns only `Dict("ode_sol" => ...)`. The ODE solution is in the interaction picture, so the lab-frame field at z=L is `uωf = cis.(fiber["Dω"] * L) .* sol["ode_sol"](L)`. This follows the same pattern used in the solver's zsave branch (line 192 of simulate_disp_mmf.jl: `uω_z[i,:,:] = cis.(fiber["Dω"] * fiber["zsave"][i]) .* sol_disp_mmf(fiber["zsave"][i])`).

**Helper function: `pab_make_ablation_window`** (D-07, D-08)
Build a smooth zeroing window on the fftshifted frequency axis. The window is 1 everywhere except in the zeroed band, where it transitions to 0 via super-Gaussian roll-off:
```julia
function pab_make_ablation_window(fs_fftshifted, band_lo, band_hi;
                                   sg_order=PAB_SG_ORDER, sg_width_frac=PAB_SG_WIDTH_FRAC)
    window = ones(length(fs_fftshifted))
    band_width = band_hi - band_lo
    sg_sigma = max(sg_width_frac * band_width, 1e-6)
    for (i, f) in enumerate(fs_fftshifted)
        if band_lo <= f <= band_hi
            dist_lo = f - band_lo
            dist_hi = band_hi - f
            margin = min(dist_lo, dist_hi)
            window[i] = 1.0 - exp(-(margin / sg_sigma)^sg_order)
        end
    end
    return window
end
```

**Helper function: `pab_shift_phase_spectrum`** (D-11)
Translate phi_opt by delta_f THz on the frequency grid using linear interpolation with zero extrapolation:
```julia
function pab_shift_phase_spectrum(phi_opt, sim, delta_f_THz)
    Nt = length(phi_opt)
    fs_shifted = fftshift(fftfreq(Nt, 1/sim["Δt"]))  # THz (fftshifted)
    phi_on_shifted_axis = fftshift(phi_opt)
    itp = LinearInterpolation(fs_shifted, phi_on_shifted_axis, extrapolation_bc=0.0)
    phi_new_shifted = itp.(fs_shifted .- delta_f_THz)
    return ifftshift(phi_new_shifted)  # back to FFT order
end
```

**Experiment 1: Band zeroing (D-07, D-08)**
For each canonical config:
1. Compute the signal band extent from the spectrum: find frequencies where `abs2(fftshift(uω0))` > -40 dB of peak (same signal_mask approach as phase_analysis.jl)
2. Divide the signal band into 10 equal-width sub-bands
3. For each sub-band k=1..10:
   a. Build ablation window on fftshifted axis using `pab_make_ablation_window`
   b. Convert window to FFT order: `window_fft = ifftshift(window_shifted)`
   c. Apply: `phi_ablated = phi_opt .* window_fft`
   d. Propagate: `J_ablated = pab_propagate_and_cost(uω0, phi_ablated, fiber, sim, band_mask)`
4. Also propagate with full phi_opt (J_full) and flat phase (J_flat) as baselines
5. Store results: for each sub-band, record (band_lo, band_hi, J_ablated, J_full, J_flat)

**Experiment 2: Cumulative ablation (D-09)**
For each canonical config:
1. Start with the full signal band sub-bands ordered from edges inward (bands 1, 10, 2, 9, 3, 8, 4, 7, 5, 6)
2. Progressively zero bands: first zero bands 1+10 (outermost), then also 2+9, then also 3+8, etc.
3. At each step, build cumulative window (product of all individual zeroing windows so far)
4. Convert to FFT order, apply to phi_opt, propagate, record J
5. Store results: sequence of (n_bands_remaining, bandwidth_remaining_THz, J_cumulative)

**Experiment 3: Global scaling (D-10)**
For each canonical config:
1. For scale_factor in [0.0, 0.25, 0.5, 0.75, 1.0, 1.25, 1.5, 2.0]:
