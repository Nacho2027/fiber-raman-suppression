# Classical Raman Suppression via Spectral Phase Shaping: Complete Findings

## Abstract
[2-3 sentences: we optimized spectral phase to suppress Raman scattering in SMF-28 and HNLF, achieving 50-77 dB suppression. Through z-resolved diagnostics, phase ablation, and multi-start analysis, we identified the mechanism as amplitude-sensitive nonlinear interference requiring sub-THz spectral alignment.]

## 1. Methods

### 1.1 Simulation Framework
- GNLSE forward propagation with Raman response (MultiModeNoise.jl)
- Adjoint-method gradient computation for L-BFGS optimization
- Cost function: J = E_Raman / E_total (linear), optimized with 10*log10(J) scaling
- Spectral phase applied at fiber input: u_shaped(omega) = u_0(omega) * exp(i * phi_opt(omega))

### 1.2 Parameter Space
- SMF-28: gamma=1.3e-3, beta2=-2.17e-26, L=0.5-5m, P=0.005-0.2W
- HNLF: gamma=1.1e-2, beta2=-0.5e-26, L=0.5-1m, P=0.005-0.03W
- Soliton number range: N=1.3 to 6.3
- Grid: Nt=8192-32768, time_window auto-sized with SPM correction

### 1.3 Analysis Techniques
- Phase decomposition: polynomial orders 2-6 (GDD through HOD)
- Phase ablation: 10-band super-Gaussian zeroing, cumulative bandwidth truncation
- Perturbation: global scaling (0-2x), spectral shift (-5 to +5 THz)
- Z-resolved diagnostics: J(z) at 50 fiber positions
- Multi-start: 10 random initial phases to probe cost landscape

[Polynomial decomposition explains only ~16% of phi_opt variance. The remaining 84% has no simple analytical form. Multi-start solutions have mean pairwise correlation 0.109.]
