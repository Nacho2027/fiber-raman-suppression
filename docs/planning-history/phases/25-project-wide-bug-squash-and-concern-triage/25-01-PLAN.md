# Phase 25 Plan 01

## Objective

Resolve the real low-risk bugs and stale canonical docs surfaced by the Phase 25 audit, verify them with the fast tier, and seed the remaining structural hazards instead of pretending they were fixed.

## Tasks

1. Audit reported concerns against live code with repo-wide grep.
2. Patch confirmed correctness bugs in live code.
3. Remove confirmed dead runtime files and update any live tooling references.
4. Add at least one regression test covering the fixed bug path.
5. Update canonical planning/docs so they describe the live code accurately.
6. Plant seeds for unresolved structural hazards.
7. Run fast-tier verification and record the result.

- Unsupported `pulse_form` values no longer fail silently.
- The dead `simulate_disp_gain_smf.jl` placeholder is removed without leaving live tooling broken.
- Canonical docs no longer advertise the deleted file as active code.
- Fast-tier tests pass in this worktree after environment instantiation.
