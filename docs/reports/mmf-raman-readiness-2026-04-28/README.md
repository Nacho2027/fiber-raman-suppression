# MMF Raman Readiness Package

Date: 2026-04-28

This folder contains the human-facing readiness package for the multimode Raman
suppression lane.

- `REPORT.md` - paper-style evidence summary, claim boundaries, figure manifest,
  and remaining paper-readiness gates.
- `PRESENTATION.md` - slide-by-slide deck outline with the exact result figures
  to show.

The generated figures and burst logs are intentionally referenced from
`results/` rather than copied into git.
