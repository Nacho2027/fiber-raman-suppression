# Scripts

Use `scripts/canonical/` for maintained commands. Use `scripts/research/` for
active research drivers that are not part of the supported surface.

| Path | Purpose |
|---|---|
| `canonical/` | user-facing wrappers |
| `lib/` | shared implementation used by scripts |
| `workflows/` | workflow bodies behind wrappers |
| `research/` | experiments and analysis drivers |
| `burst/` | burst-machine helpers |
| `ops/` | local operations helpers |
| `archive/` | old scripts kept for provenance |
